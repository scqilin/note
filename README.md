## AntV

![image-20201021114217872](README.assets/image-20201021114217872.png)

AntV 是蚂蚁金服全新一代数据可视化解决方案，致力于提供一套简单方便、专业可靠、无限可能的数据可视化最佳实践。
自吹让数据栩栩如生,不过看了几天后不得不说antv的产品文档写的有点潦草马虎。



基础产品

- G2可视化引擎
- G6图可视化引擎
- F2移动可视化方案
- L7地理空间数据可视化

扩展产品

- G2Plot开箱即用的图表库
- Graphin 基于G6封装的图分析应用组件
- CHartCube图表魔方
